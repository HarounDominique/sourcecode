## Decisions Made (Don't Re-litigate)

| Decision | Rationale |
|---|---|
| `--ci/--no-ci` flag form (not `is_flag=True`) | Avoids Typer deprecation warning in 0.24+ |
| github-comment renderer in cli.py (not spring_findings.py) | Presentation logic; spring_findings stays a pure data model |
| Score by unique file not by finding count | File with 10 javax imports = 1 file to fix; counting findings inflates penalty unfairly |
| MCP tool delegates to CLI via `_execute()` | Consistent with all other MCP tools; avoids duplicating logic |
| `when_to_call:` in docstring not separate metadata | FastMCP exposes docstring as tool description; agents read it natively |
| eureka 0/100 score is correct | Pre-Spring project, 95 affected files; score floor behavior by design |
