- `build_canonical_ir()` propagates to `cir.metadata["security_model"]`
- `project_endpoint_surface()` emits `security_model` from CIR metadata
- MCP server docstring updated with all 4 values and their semantics

**P2-10: `architecture.confidence` inconsistent between `--compact` and `--agent`** (`cli.py`)
- Root cause: `--agent` auto-enables `architecture=True` (line 908); `--compact` did not
- Fix: added `architecture = True` to the `--compact` implicit-enables block (~line 1134)
