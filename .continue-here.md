## Decisions Made (Don't Re-litigate)

| Decision | Rationale |
|---|---|
| Cache TTL 30 min (not 10 min) | 30 min = max lag for Lemon webhook detection; 10 min too aggressive for CLI startup perf |
| Device code generated client-side | Simpler — no init endpoint needed, client owns the code |
| Free delta runs = 30 (per-repo lifetime) | Simpler to track; enough for evaluation without burning quota daily |
| `--full` free up to 500 Java files | Not zero, not unlimited — matches "small repo" definition naturally |
| `defusedxml` defense-in-depth | stdlib ET can parse XXE on old Python; defusedxml is standard mitigation |
| `status != "inactive"` in `is_pro` | Handles subscription cancellation; missing field = backward compat with key-based flow |
