### CRÍTICO 1 — Truncation silent warning + --output bypasses budget
**Root cause**: `trim_to_budget` always trimmed regardless of output destination.
No stderr warning before JSON output. `_truncation_summary` field missing.
**Fix**:
- Added `skip: bool = False` param — when `--output <file>` is used, `skip=True`
  skips all budget trimming (no `_budget_note`, full output).
- Added `warn_stderr: bool = False` param — stdout path emits WARNING to stderr
  BEFORE the JSON payload when trimming occurs.
- Added `_truncation_summary.total_omitted_items` field to trimmed output.
- `cli.py` compact + agent paths pass `skip=(output is not None)` and
  `warn_stderr=(output is None)`.
**Files**: `src/sourcecode/output_budget.py`, `src/sourcecode/cli.py`
**Commit**: b962c83

### CRÍTICO 2 — prepare-context delta Pro-lock exits 1 / no free_tier_alternative
**Root cause**: `require_feature` called `sys.exit(1)`. No `free_tier_alternative`.
**Fix**:
- `license.py:require_feature` changed `sys.exit(1)` → `sys.exit(2)`.
  Convention: 0=ok, 1=runtime error, 2=license required.
- Added `extra_fields: Optional[dict]` param to `require_feature`.
- `cli.py` delta gate passes `free_tier_alternative` pointing to review-pr.
**Files**: `src/sourcecode/license.py`, `src/sourcecode/cli.py`
**Commit**: b962c83
