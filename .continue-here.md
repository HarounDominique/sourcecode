### BUG 2 — `--exclude` not propagated to `architecture_summary` / stacks [FIXED]

**Root cause:** `WorkspaceAnalyzer` builds workspace list from pre-exclusion manifests (depth-1 scan, no `_extra_excludes`). Workspace loop processed excluded workspaces → their stacks entered `sm.stacks` → `ArchitectureSummarizer` described excluded stacks.

**Fix:** `cli.py` workspace loop skips workspaces whose path parts intersect `_extra_excludes`:
```python
if _extra_excludes:
    _ws_norm = workspace.path.replace("\\", "/").strip("/")
    _ws_parts = frozenset(_ws_norm.split("/"))
    if _ws_parts & _extra_excludes:
        continue
```

### BUG 3 — CLI validation errors return plain text [FIXED]

**Root cause:** Path/flag validation used `typer.echo("Error: ...")` → plain text to stderr; AI agents/tools cannot parse.

**Fix:** Added `_emit_error_json(error, message, **context)` module-level helper in `cli.py`. Updated path-not-found, not-a-dir, `--format`, `--graph-detail`, `--docs-depth` validation to emit structured JSON. Exit codes unchanged (1 runtime, 2 usage).

JSON format: `{"error": "...", "message": "...", <context_keys>}`
