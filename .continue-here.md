### BUG FOUND AND FIXED — BUG-001 (spring_detected hardcoded)
**Commit:** `bf09082`

**Problem:** `run_tx_audit()` and `run_security_audit()` hardcoded `spring_detected=True`
regardless of actual Spring presence. Keycloak (Quarkus, 0 Spring annotations) returned
`spring_detected: True`.

**Fix in `src/sourcecode/spring_tx_analyzer.py`:**
```python
_spring_detected = tx_index.stats()["total"] > 0 or bool(model.bean_graph.beans)
```

**Fix in `src/sourcecode/spring_security_audit.py`:**
```python
_spring_detected = (
    (model is not None and bool(model.bean_graph.beans))
    or tx_index.stats()["total"] > 0
    or cir.metadata.get("security_model", "unknown") != "unknown"
)
```

**Test updated:** `test_empty_repo_no_findings` → asserts `spring_detected is False`.
2123 passed, 3 skipped. No regressions.
