### BUG 1 — fix-bug/prepare-context task cache (CRITICAL)
**Root cause**: `prepare_context_cmd` had no cache. Called `builder.build()` on
every invocation — 25s per call even with RIS warm.
**Fix**: Added task-level cache keyed on `(task, git_sha, symptom, format)`.
Cache read before `builder.build()`, cache write after output built.
Skips `delta`/`review-pr` (diff-dependent), `fast` mode, `llm_prompt` mode.
Second call completes in <1s for unchanged HEAD.
**File**: `src/sourcecode/cli.py` (~line 2566)

### BUG 2 — MCP ValidationError returns plain text, not structured JSON
**Root cause**: FastMCP's `Tool.run` catches ALL exceptions and wraps in
`ToolError(f"Error executing tool {name}: {e}")`. `pydantic.ValidationError`
from argument parsing becomes raw plain text, not structured JSON.
**Fix**: Monkey-patch `FastMCP Tool.run` at module load in `server.py`.
Catches `ToolError` wrapping a `pydantic.ValidationError`, returns structured
`{success:false, data:null, error:{code:INVALID_INPUT, message, hint}}`.
**File**: `src/sourcecode/mcp/server.py` (before `mcp = FastMCP(...)`)

### BUG 3 — endpoints ignores @M3FiltroSeguridad (99.8% none_detected)
**Root cause (a)**: `_route_security_from_sym` returned `None` when annotation
present but `nombreRecurso` value is a constant reference (no quotes). Fix:
return `{policy:custom_annotation, annotation:@M3FiltroSeguridad, resource:raw}`
**Root cause (b)**: Phase 3 of `_build_route_surface` (inheritance projection)
built route dicts without `security_annotations` key — all inherited endpoints
showed `none_detected`.
**Fix**: Phase 3 now builds `_parent_sec_by_sym` lookup from phase-2 routes
and propagates `security_annotations` to inherited routes.
**File**: `src/sourcecode/repository_ir.py`

### BUG 4 — cold-start and repo-ir output exceeds LLM context without warning
**Fix**: Both commands compute output byte size after generation.
If > 400KB (~100K tokens): emit WARNING to stderr before writing stdout.
Updated help strings document large-output behavior.
**File**: `src/sourcecode/cli.py`
