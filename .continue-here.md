### Command
```
sourcecode pr-impact --files changed_files.txt [PATH]
sourcecode pr-impact /path/to/repo --files diff.txt --format json
```

### How it works
1. Reads changed file list (one Java path per line)
2. Maps files → class FQNs via CIR node index (`_raw_ir.graph.nodes`)
3. Runs `run_impact_chain()` per modified class (reuses existing)
4. Collects event publishers/consumers from `EventGraph`
5. Collects `@Transactional` methods from `TransactionBoundaryIndex`
6. Consolidates risk + reason across all classes
7. Renders text report (default) or JSON (`--format json`)

Reuses CIR + SpringSemanticModel — no new parsers.
